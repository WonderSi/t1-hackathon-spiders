# t1-hackathon-spiders · VibeCode Jam: собеседование будущего

<div align="start">
Создайте ИИ-платформу для прохождения технических собеседований с виртуальным интервьюером.

Разработайте платформу, используя LLM инструменты на платформе Scibox, которая генерирует адаптивные задачи, предоставляет браузерную среду разработки с изоляцией и автотестами. Виртуальный интервьюер ведёт диалог, корректирует ответы и оценивает решения. Система фиксирует метрики кандидата, защищает от читерства и формирует отчёты.
